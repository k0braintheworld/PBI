# Capturas de pantalla

Coloca aquí las capturas de la interfaz para que se muestren en el `README` principal.
Usa estos nombres de archivo (PNG, ~1400 px de ancho recomendado):

| Archivo                | Qué capturar                                                        |
|------------------------|---------------------------------------------------------------------|
| `dashboard.png`        | Panel principal (resumen, calendario de copias, almacenamiento).    |
| `backups.png`          | Explorador de copias / snapshots por datastore.                     |
| `recovery.png`         | Menú de recuperación (selección de VM y punto de restauración).     |
| `jobs.png`             | Tareas programadas (prune / verify / sync).                         |
| `reports.png`          | Generación de informes (selección de fechas / VMs).                 |
| `settings-2fa.png`     | Configuración → Mi cuenta (activación de 2FA con QR).               |

Consejos:

- Inicia sesión y navega con la ventana a un tamaño consistente para que todas las
  capturas tengan la misma anchura.
- **Tapa o anonimiza** datos sensibles (IPs reales, hostnames internos, nombres de
  VMs/clientes) antes de subirlas a un repositorio público.
- Una vez añadidas, descomenta el bloque de capturas en el `README.md` principal.
